
These are ipython notebooks for the techniques in the paper. The paths are anonymized, so they would need to be updated for the user's system. 

- MLP.ipynb contains code for running the MLP model.
- GCN.ipynb contains code for running the GCN model.
- attention_model.ipynb contains code for running the cross-modal attention model and extracting attentions.
- association_rules.ipynb contains code for extracting association rules.
- ensemble.ipynb contains code for the ensemble method. Individual results are extracted and then combined. 


Required packages are listed in packages.txt.
